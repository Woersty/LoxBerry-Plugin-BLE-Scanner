Suchen Schaltfläche
Die Schaltfläche mit der Lupe aktualisiert den Status der Tags (rechts neben dem Mülleimer) und sucht bisher unbekannte Tags.

Damit können Tags aus der Konfiguration entfernt werden.
Nur für defekte oder verlorene Tags. Denn sind die Tags jedoch in Reichweite des Scanners, werden sie automatisch wieder hinzugefügt.


Statusanzeige des Tags (manuell):
Rot = Nicht in Reichweite
Grün = In Reichweite
Hinweis: Wird nur nach einer Suche aktualisiert. Nicht in Echtzeit.

Tag-Verwendung:
= Tag verwenden
= Tag nicht verwenden
Wird kein Häkchen gesetzt, werden für diesen Tag keine Daten vom Plugin ausgegeben.

Name/Kommentar
Hier kann ein Name oder Kommentar eingegeben werden. (Nur a-z A-Z 0-9 -_ )

[BLE_xx_xx_xx_xx_xx_xx]
Hier wird die Bluetooth-MAC-Adresse des TAGs angezeigt.

Miniserver-Block
Zur Festlegung, ob die Daten des Tags (in Reichweite oder nicht) an den jeweiligen Miniserver gesendet werden sollen.
