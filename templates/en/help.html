Search Button
The Search Button with the magnifying glass icon refreshes the status of the Tags (next to the bin) and searches currently unknown Tags.

To remove Tags from the current configuration.
To be used for defective of lost Tags only. If a Tag is in the scanner range, it will be added automatically during next scan.


Statusdisplay of a Tag (manual):
Red = Out of range
Green = Within radio range
Note: Refreshed after a search only. Not a realtime value.

Tag-Usage:
= Use Tag
= Ignore Tag
If unchecked, the plugin will process no data for this Tag.

Name/Comment
To enter a name or comment. (a-z A-Z 0-9 -_ only)

[BLE_xx_xx_xx_xx_xx_xx]
Displays the Bluetooth-MAC-Address of the TAGs.

Miniserver-Block
To define, if the data of the Tag (in radio range of not) is send to the respective Miniserver.
